01

한눈에

AI 도구만으로 되는 일이 아닙니다. AI를 다뤄온 VDVIA가 우정님의 반복 업무(기획서 검수 → 제작 의뢰 → 업로드)를 효율화 · 자동화하고, 사람의 판단이 필요한 두 곳(법규 검수 · 최종 승인)은 그대로 둡니다.

  • 3필요 데이터 · 계정 권한 위임 범위

필요한 것

브랜드 · 법규 · 감도 데이터와 계정 권한을 한 번 받아 시스템에 학습 · 세팅. 이후 우정님 업무는 검수로 단순화.

왜 VDVIA인가 — 핵심은 AI 도구가 아니라 AI를 다루는 역량입니다. VDVIA는 자사 브랜드에서 콘텐츠 자동 생성 · 발행 파이프라인과 성과 트래킹 시스템을 직접 구축 · 운용해 왔고, 그 노하우를 그룹 브랜드인 TESS · ZISHEL XOMAGE 운영에 함께 적용합니다.
02

운영 구조

각 단계의 AI는 VDVIA가 세팅 · 운용 · 교정합니다 — 자동이되 방치가 아닙니다. 두 개의 검수 게이트가 핵심으로, ① AI 사전 검수는 명백한 위반을 생성 단계에서 거르고, ② 우정님 최종 검수는 법적 책임선으로 반드시 유지합니다.

입력 · 적재 AI 학습 · 생성 검수 게이트 발행 자동화
범위 = 업로드 자동화까지 · 광고비 · 전환 트래킹은 범위 밖 (코스메틱 브랜드만 예외).  발행 자동화 연동(Zapier 등)은 메타 계정 구조 확인 후 확정합니다.
03

04

필요 데이터 — 우정님은 6가지만

우정님이 새로 만드실 건 거의 없습니다. 대부분 이미 보유하신 자료를 한 번 공유해주시면 — 학습 · 룰 세팅부터 콘텐츠 생성 · 발행 자동화까지 나머지는 VDVIA가 맡습니다. (발행은 계정별 주 2건 기준)

①우정님께 선제 요청 — 6가지

1
주력 국가 + 올해 공식 이벤트
중요한 날(예: 크리스마스)에 콘텐츠를 올리고 싶은 국가들 + 브랜드별 올해 공지 필요 이벤트(부스 행사 등) 선공유
2
메타 계정 정보
발행 자동화 연동의 기반 — 연동 방식은 받은 뒤 VDVIA가 확인
3
브랜드 제품별 마케팅 덱
제품명 · 슬로건 · 핵심 소구점 · 세부 원료 · 효능 · 타깃 페르소나 · 표현 금기/필수표기 · 인증/클레임 · 브랜드 스토리 · 해시태그 · 멘션 세트
4
검수 체크리스트 + 반려 사례
기존 활용하시던 공정위 · 메디컬 · 금지어 가이드 기반 — AI 사전 검수 룰의 시드
5
원본 사진 · 영상 + 전 SKU 누끼
콘텐츠 제작 원재료 (원본만 주시면 누끼는 VDVIA가 제작 가능)
6
기존 영업팀 기획서 양식
AI 사전 검수 · 자동 이관의 입력 스키마

②VDVIA가 준비 — 받은 6개로 자체 제작 · 처리

A학습 · 검수 룰

  • 마케팅 덱 · 검수 체크리스트 · 기획서 양식을 AI가 학습/검수 가능한 형태로 구조화
  • 공정위 · 메디컬 · 금지어 AI 사전 검수 가이드라인 템플릿 구축 (체크리스트 · 반려사례 + 법령 보강)
  • 5개국 규제 · 필수표기 · 디스클로저 · 민감일 · 플랫폼 정책 법규 리서치 보강
  • 브랜드 보이스 · 톤 가이드 — 마케팅 덱 + 기존 SNS 게시물에서 추출

B콘텐츠 설계 · 생성

  • 콘텐츠 유형 4~5종 확립
  • 유형 × 포맷 매핑 · 캡션 규칙 · 해시태그 운용 규칙(받은 세트 기반) · CTA · 링크 · 주간 루틴(계정별 주 2건) · 시각 규격 · 디자인 템플릿
  • 감도 학습 — 기존 브랜드 SNS에서 합격 후보 선별
  • 전 SKU 누끼 제작, 이미지 · 영상 생성 · 편집

C캘린더 · 스케줄

  • 5개국 공휴일 · 기념일 12개월 자동 수집, 캘린더 색상 · 우선순위 · 타임존 · 게시 최적시각 · 게시 언어 정책 제안
  • 받은 공식 이벤트를 캘린더에 고정 슬롯으로 편성

D파이프라인 · 자동화

  • 기획서 → AI 사전 검수 → 캘린더/폴더 자동 이관 흐름
  • Drive 폴더 구조 · OK 신호 규약 설계, 발행 자동화 연동(메타 정보 후 방식 확정)
  • 기존 디자인 리퀘스트 · 캘린더 툴 통합 · 개선

E운영 · 법무 문서

  • 검수 책임 · 승인 합의서 초안 작성
  • 검수 SLA · 발행 일정 · 담당자 연락 체계 제안
05

대행 범위

이번 건은 콘텐츠 제작 + 업로드 자동화까지입니다. 광고 성과 운영은 별개 영역으로 둡니다.

✓ 범위 안

  • 기획서 AI 사전 검수 및 캘린더 · 폴더 자동 이관
  • 콘텐츠 제작 · 생성 (이미지 · 카드 · 영상 편집)
  • 5개국 캘린더 기반 정기 + 행사 콘텐츠
  • 현지 파트너 계정 스토리 공유 자동 처리
  • 콘텐츠 발행 AI 자동화 (검수 OK → 자동 게시)
  • 기존 디자인 리퀘스트 · 캘린더 툴 통합 · 개선

✕ 범위 밖

  • 광고비 · 전환 · ROI 트래킹 — 회사 SNS 계정은 업로드까지만 (코스메틱 브랜드만 예외)
  • POP · 패키징 등 오프라인 제작물 — 디자인팀 직접 컨택, 우정님 검수
  • 영상 variation 자동화 — 1차 결과물 확인 후 범위 결정
06

데이터 수령과 운영 방향 정렬을 먼저 끝내고, 셋업 · 검증을 거쳐 착수합니다. 목표 착수 시점은 약 2주 후입니다.

SNS 운영 방향 재정립

디자인팀 합류 미팅

SNS 운영 방식 · 모니터링 방법을 디자인팀과 함께 확정.

병행

Zapier ↔ 메타 연동 확인

메타 계정 구조가 복잡 — 예약 발행 자동화 가능 여부를 별도 검증.

1주차

데이터 수령 (P0 게이트)

5개국 · 운영단위 · 계정권한 · 검수합의서 + 브랜드 · 법규 · 감도 자산 확보.

2주차

학습 · 파이프라인 셋업 + 검증

가이드라인 템플릿 · 생성 · 검수 · 발행 연결, 1차 결과물 검수.

~2주 후

운영 착수

정기 발행 시작. 준비 상황에 따라 시점 조정 가능.

확인이 필요한 사항

발행 자동화 연동 방식

메타 계정 정보 수령 후 VDVIA가 연동 방식을 확인합니다. 제약이 있으면 발행 자동화 방식을 조정 — 클라이언트 추가 결정 사항 아님.

현지어 검수 인력

우정님이 직접 읽기 어려운 언어(아랍어 · 러시아어 등)로 게시 시 법무 검수 책임 주체 합의 필요.

성공 기준(KPI)

회사 계정은 운영 · 발행 효율 중심. 코스메틱 브랜드는 별도 전환 지표 적용 여부 협의.

NDA · 권한 회수 규약
