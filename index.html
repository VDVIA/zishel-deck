01

한눈에

AI 도구만으로 되는 일이 아닙니다. AI를 다뤄온 VDVIA가 우정님의 반복 업무(기획서 검수 → 제작 의뢰 → 업로드)를 효율화 · 자동화하고, 사람의 판단이 필요한 두 곳(법규 검수 · 최종 승인)은 그대로 둡니다.

필요한 것

브랜드 · 법규 · 감도 데이터와 계정 권한을 한 번 받아 시스템에 학습 · 세팅. 이후 우정님 업무는 검수로 단순화.

왜 VDVIA인가 — 핵심은 AI 도구가 아니라 AI를 다루는 역량입니다. VDVIA는 자사 브랜드에서 콘텐츠 자동 생성 · 발행 파이프라인과 성과 트래킹 시스템을 직접 구축 · 운용해 왔고, 그 노하우를 그룹 브랜드인 TESS · ZISHEL XOMAGE 운영에 함께 적용합니다.
02

운영 구조

각 단계의 AI는 VDVIA가 세팅 · 운용 · 교정합니다 — 자동이되 방치가 아닙니다. 두 개의 검수 게이트가 핵심으로, ① AI 사전 검수는 명백한 위반을 생성 단계에서 거르고, ② 우정님 최종 검수는 법적 책임선으로 반드시 유지합니다.

입력 · 적재 AI 학습 · 생성 검수 게이트 발행 자동화
범위 = 업로드 자동화까지 · 광고비 · 전환 트래킹은 범위 밖 (코스메틱 브랜드만 예외).  발행 자동화 연동(Zapier 등)은 메타 계정 구조 확인 후 확정합니다.

04

VDVIA가 요청하는 데이터

자동화 품질은 학습 데이터 품질에서 나옵니다. 아래를 한 번 받으면 시스템에 세팅하고, 이후 우정님 업무는 크게 줄어듭니다. P0은 착수 필수입니다.

🔑 가장 먼저 정해야 할 4가지 — 이게 안 정해지면 나머지가 묶입니다

  1. 핵심 5개국 확정 + 선정 근거법규 · 캘린더 · 언어 · 페르소나 · 파트너 매핑이 전부 여기에 달려 있습니다.
  2. 운영 단위 — 계정 · 브랜드별(TESS / ZISHEL XOMAGE / 그룹) 주당 게시 건수생성량 · 계정 라우팅이 2배로 갈리는 분기값입니다.
  3. 메타 계정 구조 · 권한 정보발행 자동화 연동 설계의 기반입니다. 받는 즉시 VDVIA가 연동 방식을 확인합니다.
  4. 검수 책임 합의서 (OK = 게시 승인 = 법적 책임 지쉘그룹 귀속)다국가 · 다국어 자동 발행을 켜기 위한 법적 선결 조건입니다.

05

대행 범위

이번 건은 콘텐츠 제작 + 업로드 자동화까지입니다. 광고 성과 운영은 별개 영역으로 둡니다.

✓ 범위 안

  • 기획서 AI 사전 검수 및 캘린더 · 폴더 자동 이관
  • 콘텐츠 제작 · 생성 (이미지 · 카드 · 영상 편집)
  • 5개국 캘린더 기반 정기 + 행사 콘텐츠
  • 현지 파트너 계정 스토리 공유 자동 처리
  • 콘텐츠 발행 AI 자동화 (검수 OK → 자동 게시)
  • 기존 디자인 리퀘스트 · 캘린더 툴 통합 · 개선

✕ 범위 밖

  • 광고비 · 전환 · ROI 트래킹 — 회사 SNS 계정은 업로드까지만 (코스메틱 브랜드만 예외)
  • POP · 패키징 등 오프라인 제작물 — 디자인팀 직접 컨택, 우정님 검수
  • 영상 variation 자동화 — 1차 결과물 확인 후 범위 결정

회사 계정은 운영 · 발행 효율 중심. 코스메틱 브랜드는 별도 전환 지표 적용 여부 협의.
