04

VDVIA가 요청하는 데이터

자동화 품질은 학습 데이터 품질에서 나옵니다. 아래를 한 번 받으면 시스템에 세팅하고, 이후 우정씨 업무는 크게 줄어듭니다. P0은 착수 필수입니다.

🔑 가장 먼저 정해야 할 4가지 (이게 안 정해지면 나머지가 묶입니다)

  1. 핵심 5개국 확정 + 선정 근거법규·캘린더·언어·페르소나·파트너 매핑이 전부 여기에 달림
  2. 운영 단위 — 계정·브랜드별(TESS / ZISHEL XOMAGE / 그룹) 주당 게시 건수생성량·계정 라우팅이 2배로 갈리는 분기값
  3. 메타 계정 구조 + Zapier 연동 가능여부예약 발행 자동화의 최대 변수 — 별도 확인 필요
  4. 검수 책임 합의서 (OK = 게시 승인 = 법적 책임 클라이언트 귀속)다국가·다국어 자동발행을 켜기 위한 법적 선결 조건

대행 범위

이번 건은 콘텐츠 제작 + 업로드 자동화까지입니다. 광고 성과 운영은 별개 영역으로 둡니다.

✓ 범위 안

  • 기획서 AI 사전검수 및 캘린더·폴더 자동 이관
  • 콘텐츠 제작·생성 (이미지·카드·영상 편집)
  • 5개국 캘린더 기반 정기 + 행사 콘텐츠
  • 현지 파트너 계정 스토리 공유 자동 처리
  • Zapier 예약 발행 (6~12개월 스케줄)
  • 기존 디자인 리퀘스트·캘린더 툴 통합·개선

✕ 범위 밖

  • 광고비·전환·ROI 트래킹 — 회사 SNS 계정은 업로드까지만 (코스메틱 브랜드만 예외)
  • POP·패키징 등 오프라인 제작물 — 디자인팀 직접 컨택, 우정씨 검수
  • 영상 variation 자동화 — 1차 결과물 확인 후 범위 결정
06

일정 · 다음 단계

데이터 수령과 운영 방향 정렬을 먼저 끝내고, 셋업·검증을 거쳐 착수합니다. 목표 착수 시점은 약 2주 후입니다.

월요일

SNS 운영 방향 재정립

우정씨 복귀 후 포스팅 방식·모니터링 방법을 겨레님과 정리.

화요일

디자인팀 합류 미팅

SNS 운영 방식·모니터링 방법을 디자인팀과 함께 확정.

병행

Zapier ↔ 메타 연동 확인

메타 계정 구조가 복잡 — 예약 발행 자동화 가능여부를 별도 검증.

1주차

데이터 수령 (P0 게이트)

5개국·운영단위·계정권한·검수합의서 + 브랜드·법규·감도 자산 확보.

2주차

학습 · 파이프라인 셋업 + 검증

가이드라인 템플릿·생성·검수·발행 연결, 1차 결과물 검수.

~2주 후

운영 착수

정기 발행 시작. 레몬더마 SNS 준비 상황에 따라 시점 조정 가능.

확인이 필요한 사항

Zapier ↔ 메타 계정 연동 가능여부

안 될 경우 예약 발행 설계가 바뀜 — 담당자 지정 후 우선 확인.

현지어 검수 인력

우정씨가 못 읽는 언어(아랍어·러시아어 등)로 게시 시 법무 검수 책임 주체 합의 필요.

성공 기준(KPI)

회사 계정은 운영·발행 효율 중심. 코스메틱 브랜드는 별도 전환 지표 적용 여부 협의.

NDA · 권한 회수 규약

전성분·미공개 제품·계정 권한 위임 — 보관/파기·종료 시 회수 절차 합의.
