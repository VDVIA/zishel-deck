3잘 되는 것

  • 텍스트 렌더링이 예상보다 강력. 이벤트/웨비나 카드의 날짜·시간·연자명·헤드라인이 정확히 재현 — 과거 테스트에서 나던 "없는 단어 지어내기(환각)"가 이번엔 거의 사라짐.
  • 실제 바이알 누끼 투입 = 돌파구. 깨끗한 제품 누끼를 레퍼런스로 넣으면 제품 형태·큰 라벨 텍스트가 거의 완벽 재현. 실물 그대로의 브랜드 정체가 유지됨.
  • 모델 에디토리얼(제품+모델). 실제 모델 촬영본을 넣으니 웻룩 헤어·글래스 스킨·여백 감도가 실제 캠페인 프레임 수준으로 나옴 — 신규 컷을 계속 뽑을 수 있는 잠재력.
  • 브랜드 팔레트·구도 일관성. 티파니-민트 + 화이트 + 글로시 리퀴드 톤을 안정적으로 유지 — 유형 전반 충실도 78~88%.

4아쉬운 것

  • 초소형 라벨 텍스트. 바이알 최하단 미세 문구("CELLULAR TRIGGER (FECT) COMPLEX")가 "CONFLEX" 등으로 미세하게 깨짐 — 썸네일에선 티 안 나지만 B2B 메디컬 정밀 기준엔 미달. → 실제 라벨 오버레이 고정 공정 필요.
  • 간헐적 개별 실패. 4컷 중 1컷 정도는 라벨 좌우반전·글자 뭉개짐 등으로 폐기 → 매 컷 사람 QA 게이트가 필수(무보정 자동발행 불가).
  • 색 과채도. 무드/일부 컷이 레퍼런스의 절제된 파스텔 티파니보다 채도가 높게 나옴 → 간단한 색보정 필요.
  • 텍스트 밀도가 높을수록 취약. 큰 워드마크·짧은 문구는 잘 되지만, 문단형 조밀 텍스트(마스크팩 성분 설명 등)는 깨짐 — 조밀 텍스트는 사람이 얹는 게 안전.
  • 모델 초상권. 실제 모델을 AI가 알아볼 수 있게 재현 → 발행 전 초상 사용 동의(AI 생성물 포함) 확인 필요.
